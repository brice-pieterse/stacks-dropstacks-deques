## About

This project explores the implementation of an arraystack, drop out stack, and double ended queue. 

Exhibits of logs can be found in the public folder. Those logs should align with the comments left in the driver classes indicating the current state of the data structure. 

### Testing ArrayStack

 ```javac -cp . arraystack/ArrayStackDriver.java```
 ```java arraystack/ArrayStackDriver```


### Testing DropStack

 ```javac -cp . dropstack/DropOutStackDriver.java```
 ```java dropstack/DropOutStackDriver```


### Testing Deque

 ```javac -cp . deque/DequeDriver.java```
 ```java deque/DequeDriver```
